# gen-ui-template

This is a Next.js application generated with
[Create Netronk Gen](https://github.com/netronk/gen).

Run development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
